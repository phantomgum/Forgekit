# TrendGenie (Idea Generator)

A modern, Linear-inspired AI project generator.

## UI Redesign
- Scraped Linear using Firecrawl to extract brand guidelines.
- Switched to the Inter font for a clean, modern aesthetic.
- Updated color palette to an ultra-dark theme (almost pure black background `#08090A`) with subtle radial gradients.
- Added deep contrast between cards (`#121315`) and the background, giving the cards a "floating" feel.
- Modified buttons to match a sleek capsule-like pill aesthetic.
- Styled input forms to have beautiful active states and smooth transitions.
- Replaced the simple loader with a high-end skeleton shimmer effect.
